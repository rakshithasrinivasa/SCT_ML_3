# 🐱🐶 Cats vs Dogs Image Classification using SVM

## 📌 Task
SkillCraft Technology - Machine Learning Internship

### Task 03: Implement a Support Vector Machine (SVM) to classify images of cats and dogs.

## 📖 Project Overview
This project implements a Support Vector Machine (SVM) classifier to distinguish between cat and dog images. The images are preprocessed by resizing them and converting them into feature vectors before training the model.

## 🛠️ Technologies Used
- Python
- Google Colab
- OpenCV
- NumPy
- Scikit-learn

## 📂 Dataset
- Microsoft Cats vs Dogs Dataset (Kaggle)

## 🚀 Steps Performed
- Downloaded the dataset from Kaggle.
- Preprocessed images by resizing them to **64 × 64** pixels.
- Flattened the images into feature vectors.
- Split the dataset into training and testing sets.
- Trained a **Linear Support Vector Machine (SVM)** classifier.
- Evaluated the model using **Accuracy** and **Classification Report**.
- Predicted the class of sample images.

## 📊 Results
- Successfully classified cat and dog images using SVM.
- Achieved approximately **56% accuracy** on the test dataset.
